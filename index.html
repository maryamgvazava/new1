### **Step 1: Divide 10.0.0.0/8 Network into Subnets**
We divide the 10.0.0.0/8 network into 4 subnets:
1. **Subnet 1 (VLAN 50 & VLAN 60 - IT & HR)** → 10.0.0.0/10
2. **Subnet 2 (VLAN 70 & VLAN 80 - Sales & LAW)** → 10.64.0.0/10
3. **Subnet 3 (Serial Link between R1 & R2)** → 192.168.20.0/24
4. **Subnet 4 (Other Use)** → 10.128.0.0/10
#### Example Calculation:
- Subnet 1: 10.0.0.0/10
- Network Address: 10.0.0.0
- Subnet Mask: 255.192.0.0 (10.0.0.0 to 10.63.255.255)
- Subnet 2: 10.64.0.0/10
- Network Address: 10.64.0.0
- Subnet Mask: 255.192.0.0 (10.64.0.0 to 10.127.255.255)
- Subnet 3: 192.168.20.0/24
- Network Address: 192.168.20.0
- Subnet Mask: 255.255.255.0
- Subnet 4: 10.128.0.0/10
- Network Address: 10.128.0.0
- Subnet Mask: 255.192.0.0 (10.128.0.0 to 10.191.255.255)
### **Step 2: Assign Hostnames to Devices**
1. **On Routers (R1, R2)**:
``` R1(config)# hostname R1
R2(config)# hostname R2
``` 2. **On Switches (S1, S2, S3, S4, S5, S6)**:
``` S1(config)# hostname S1
S2(config)# hostname S2
S3(config)# hostname S3
S4(config)# hostname S4
S5(config)# hostname S5
S6(config)# hostname S6
``` ### **Step 3: Configure VLANs on Switches**
- **For VLAN 50 (IT) and VLAN 60 (HR) on switches S1, S2, and S5**:
``` S1(config)# vlan 50
S1(config-vlan)# name IT
S1(config)# vlan 60
S1(config-vlan)# name HR
S2(config)# vlan 50
S2(config-vlan)# name IT
S2(config)# vlan 60
S2(config-vlan)# name HR
S5(config)# vlan 50
S5(config-vlan)# name IT
S5(config)# vlan 60
S5(config-vlan)# name HR
``` - **For VLAN 70 (Sales) and VLAN 80 (LAW) on switches S3, S4, and S6**:
``` S3(config)# vlan 70
S3(config-vlan)# name Sales
S3(config)# vlan 80
S3(config-vlan)# name LAW
S4(config)# vlan 70
S4(config-vlan)# name Sales
S4(config)# vlan 80
S4(config-vlan)# name LAW
S6(config)# vlan 70
S6(config-vlan)# name Sales
S6(config)# vlan 80
S6(config-vlan)# name LAW
``` ### **Step 4: Configure Subinterfaces for VLANs on Routers**
- **For R1:**
``` R1(config)# interface GigabitEthernet0/0.50
R1(config-subif)# encapsulation dot1Q 50
R1(config-subif)# ip address 10.0.0.1 255.255.255.192
R1(config-subif)# no shutdown
R1(config)# interface GigabitEthernet0/0.60
R1(config-subif)# encapsulation dot1Q 60
R1(config-subif)# ip address 10.0.0.65 255.255.255.192
R1(config-subif)# no shutdown
``` - **For R2:**
``` R2(config)# interface GigabitEthernet0/0.70
R2(config-subif)# encapsulation dot1Q 70
R2(config-subif)# ip address 10.64.0.1 255.255.255.192
R2(config-subif)# no shutdown
R2(config)# interface GigabitEthernet0/0.80
R2(config-subif)# encapsulation dot1Q 80
R2(config-subif)# ip address 10.64.0.65 255.255.255.192
R2(config-subif)# no shutdown
``` ### **Step 5: Configure Serial Interface Between R1 and R2**
- **For R1 (Serial 0/0/0):**
``` R1(config)# interface Serial0/0/0
R1(config-if)# ip address 192.168.20.1 255.255.255.0
R1(config-if)# clock rate 64000
R1(config-if)# no shutdown
``` - **For R2 (Serial 0/0/0):**
``` R2(config)# interface Serial0/0/0
R2(config-if)# ip address 192.168.20.2 255.255.255.0
R2(config-if)# no shutdown
``` ### **Step 6: Configure DHCP on R2**
- Exclude IP range for DHCP:
``` R2(config)# ip dhcp excluded-address 10.0.0.1 10.0.0.10
``` - **For VLAN 50**:
``` R2(config)# ip dhcp pool VLAN50
R2(dhcp-config)# network 10.0.0.0 255.255.255.192
R2(dhcp-config)# default-router 10.0.0.1
``` - **For VLAN 60**:
``` R2(config)# ip dhcp pool VLAN60
R2(dhcp-config)# network 10.0.0.64 255.255.255.192
R2(dhcp-config)# default-router 10.0.0.65
``` - **For VLAN 70**:
``` R2(config)# ip dhcp pool VLAN70
R2(dhcp-config)# network 10.64.0.0 255.255.255.192
R2(dhcp-config)# default-router 10.64.0.1
``` - **For VLAN 80**:
``` R2(config)# ip dhcp pool VLAN80
R2(dhcp-config)# network 10.64.0.64 255.255.255.192
R2(dhcp-config)# default-router 10.64.0.65
``` ### **Step 7: Configure Trunk Ports on Switches**
- For inter-switch and router-to-switch connections:
``` S1(config)# interface GigabitEthernet0/1
S1(config-if)# switchport mode trunk
S2(config)# interface GigabitEthernet0/1
S2(config-if)# switchport mode trunk
S5(config)# interface GigabitEthernet0/1
S5(config-if)# switchport mode trunk
S3(config)# interface GigabitEthernet0/1
S3(config-if)# switchport mode trunk
S4(config)# interface GigabitEthernet0/1
S4(config-if)# switchport mode trunk
S6(config)# interface GigabitEthernet0/1
S6(config-if)# switchport mode trunk
``` ### **Step 8: Configure Access Ports for PCs**
- **For PC connections in VLAN 50, 60, 70, 80**:
``` S1(config)# interface FastEthernet0/1
S1(config-if)# switchport mode access
S1(config-if)# switchport access vlan 50
S2(config)# interface FastEthernet0/1
S2(config-if)# switchport mode access
S2(config-if)# switchport access vlan 60
S3(config)# interface FastEthernet0/1
S3(config-if)# switchport mode access
S3(config-if)# switchport access vlan 70
S4(config)# interface FastEthernet0/1
S4(config-if)# switchport mode access
S4(config-if)# switchport access vlan 80
``` ### **Step 9: Enable SSH on Routers**
- **For R1 and R2:**
``` R1(config)# username admin privilege 15 secret cisco
R1(config)# ip domain-name cisco.com
R1(config)# crypto key generate rsa
R1(config)# ip ssh version 2
R1(config)# line vty 0 4
R1(config-line)# login local
R1(config-line)# transport input ssh
R2(config)# username admin privilege 15 secret cisco
R2(config)# ip domain-name cisco.com
R2(config)# crypto key generate rsa
R2(config)# ip ssh version 2
R2(config)# line vty 0 4
R2(config-line)# login local
R2(config-line)# transport input ssh
``` ### **Step 10: Set Console and Privileged Mode Passwords**
- **For R1 and R2:**
``` R1(config)# line console 0
R1(config-line)# password class
R1(config-line)# login
R1(config)# enable secret cisco
R2(config)# line console 0
R2(config-line)# password class
R2(config-line)# login
R2(config)# enable secret cisco
``` ### **Step 11: Encrypt Passwords**
- **For R1 and R2:**
``` R1(config)# service password-encryption
R2(config)# service password-encryption
``` ### **Step 12: Configure DNS on Server**
- On Server:
``` Server(config)# ip dns server
Server(config)# ip host cisco.com 10.0.0.100
``` ### **Step 13: Restrict PC3 from Web Service Using ACL**
- On R2:
``` R2(config)# access-list 100 deny tcp host 10.0.0.3 any eq 80
R2(config)# access-list 100 permit ip any any
R2(config)# interface GigabitEthernet0/0
R2(config-if)# ip access-group 100 in
``` ### **Step 14: Backup Router Configurations to TFTP**
- **For R1:**
``` R1# copy running-config tftp
Enter TFTP server address: 10.0.0.200
``` - **For R2:**
``` R2# copy running-config tftp
Enter TFTP server address: 10.0.0.200
``` ### **Final Step: Save Configuration**
- **For both routers:**
``` R1# write memory
R2# write memory
``
` This provides the full configuration based on your setup.
